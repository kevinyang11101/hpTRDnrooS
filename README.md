# 仓库管理系统 java7

## 项目概述

仓库管理系统是一款基于B/S架构的开发项目，覆盖了企业管理链路中的客户、订单、生产、待检、入库、盘点及出库等多个环节。该项目采用前后端分离的技术架构，后端使用Spring Boot、Spring Security、MyBatis Plus、Redis及WebSocket等技术栈；前端则基于Vue、ElementUI、ECharts、Vuex和WebSocket构建。

## 功能模块

- **权限管理**：对系统用户的权限进行管理。
- **用户管理**：维护系统用户信息。
- **角色列表**：定义不同角色及其权限。
- **订单管理**：处理订单相关信息。
- **仓库管理**：管理仓库存储情况。
- **出库管理**：管理产品出库流程。
- **入库管理**：管理产品入库流程。
- **客户管理**：维护客户资料。
- **统计管理**：提供数据统计与分析。
- **聊天室**：集成即时通讯功能。

## 系统角色

- 系统管理员
- 普通用户
- 客户
- 生产人员
- 仓库管理员

## 运行环境

- 后端：Java 1.8+，Spring Boot 2.x，Redis 5.x
- 前端：Vue 2.x，Node.js 10.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 准备后端运行环境，安装Java、MySQL、Redis等。
2. 部署后端服务，通过Spring Boot命令或容器编排工具启动服务。
3. 部署前端应用，构建Vue项目并放置于Web服务器。
4. 配置WebSocket服务，确保即时通讯功能正常。
